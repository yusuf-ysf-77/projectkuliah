# owTask Theme Redesign Plan

## Overview

Redesign owTask dari tema amber/forest-green menjadi tema modern, playful, vibrant bergaya Nunito + DeDo. Primary color: hijau lime/ceriah dengan gradient. Sidebar: terang (white/light) dengan gradient accents. Multi-color decorative elements.

---

## Phase 1: Theme Tokens & Utilities (`app/globals.css`)

### New Color System

```
Light Mode:
  --background: #F5F7FA        (cool light gray, bukan warm)
  --foreground: #1A1A2E        (deep navy-black)
  --card: #FFFFFF
  --primary: #22C55E           (green-500, fresh green)
  --primary-foreground: #FFFFFF
  --secondary: #F0FDF4         (green-50)
  --accent: #8B5CF6            (violet-500, multi-color accent)
  --muted: #F1F5F9             (slate-100)
  --border: #E2E8F0            (slate-200)
  --sidebar: #FFFFFF           (light sidebar!)
  --sidebar-foreground: #1A1A2E
  --sidebar-primary: #22C55E
  --sidebar-accent: #F0FDF4
  --sidebar-border: #E2E8F0
```

### New/Updated Utility Classes

- `.gradient-primary` → green lime gradient: `linear-gradient(135deg, #22C55E, #16A34A)`
- `.gradient-hero` → cool light gradient (not warm)
- `.gradient-text` → green gradient text
- `.gradient-sidebar` → white-to-light-green subtle gradient
- `.gradient-card` → subtle multi-color shimmer on cards
- `.bg-warm` → rename/replace to `.bg-cool` (cool gray gradient)
- `.glass-card` → update border to more translucent white
- Keep `.glass`, `.glass-strong`, `.glass-sidebar` (updated)
- New: `.decoration-blob` for colorful floating decorative shapes
- New: `.dashed-line` for connecting line decorations
- New: `.sparkle` animation for star decorations

### New Keyframe Animations

- `@keyframes sparkle` — pulse opacity for star decorations
- `@keyframes blob` — slow morph/drift for decorative blobs
- `@keyframes dash` — dash offset animation for dashed lines
